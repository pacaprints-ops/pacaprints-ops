# Wedding Planner — Testing Notes

Notes from Vicky for Carrie to review.

## Initial Feedback

- **Budget** — needs to show whether the booking is under budget
- **Date** — the date field should display the date of purchase, not just the event date
- **Payment options** — needs to support options such as deposit, payment plans, etc.

## Testing Notes — Round 2

1. **Dashboard — Getting Started numbering** — steps are numbered 2, 3, 4, 6 — number 5 is missing, needs to be in order
2. **Getting Started — section order** — "Fill in your website" section should be moved to last in the Getting Started list
3. **Budget / Finance tab** — when adding a spend, deposit or paid in full, there should be a separate section (or finance tab) showing: what has been paid, how much is still due, and the due dates
4. **Quotes — edit option** — once a quote has been added, there should be an option to edit it rather than having to delete and re-add if a mistake was made
5. **Suppliers — booking without quotes** — currently you have to add a quote before you can confirm a booking (which then adds to the budget). Need an option to book a supplier directly without going through quotes first
6. **Tables — delete button size** — the "Delete this table" button is too small and hard to find, needs to be more prominent
7. **Tables — evening guests** — the tables section counts evening guests as unseated. Evening guests shouldn't be included in the seated count as they don't need a table place
8. **Day plan — schedule edit option** — when a schedule item has been added, there should be an edit/amend option. Currently you have to delete and re-add if there's an error
9. **Dashboard — budget totals** — total budget and remaining budget are blank on the dashboard even after all suppliers and spend have been entered. These figures show correctly in the budget tab but should also be visible on the dashboard
